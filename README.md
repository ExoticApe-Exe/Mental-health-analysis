## 🚀 Project Overview

This project explores **mental health trends in the tech industry** using real-world survey data.
It focuses on identifying how **demographics, workplace environment, and lifestyle factors** influence mental health conditions and treatment behavior.

📊 The project combines:

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Advanced Data Visualization
* Interactive Charts
* Machine Learning Modeling

---

## 🎯 Objectives

* Analyze mental health patterns across different demographics
* Understand the impact of workplace conditions
* Identify key factors influencing treatment seeking behavior
* Build a predictive model for mental health treatment

## 🔍 Key Insights

* 📌 Employees in high-stress environments are more likely to seek treatment
* 📌 Work interference strongly correlates with mental health conditions
* 📌 Younger professionals show higher mental health concerns
* 📌 Gender differences influence treatment-seeking behavior

## 🤖 Machine Learning

A **Random Forest Classifier** was implemented to predict whether an individual is likely to seek mental health treatment based on various factors.
